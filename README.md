# 🌐 Personal Portfolio Website

## 📌 Features

* ✨ Modern and responsive design
* 🎯 Smooth scrolling navigation
* ⌨️ Typing animation in the Hero section
* 👨‍💻 About Me section
* 🎓 Education Timeline
